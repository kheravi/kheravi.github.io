Database Systems Course History

CSE 232A Graduate Database Systems |UCSD|
This course covers key principles and systems design issues in RDBMSs such as: storage management, query processing and optimization, transaction management, concurrency control, recovery, parallel DBMSs, and dataflow systems.

CSE 232B Database Systems: Advanced Topics and Implementation |UCSD|
A hands-on approach to the principles of databases implementation. Algebraic rewriters/optimizers, query processors, triggers. Beyond centralized relational databases. For this course, the main projects revolved around writing an engine for XPath, a sub language of XQuery. I implemented a tree structure of visitors similar to writing a compiler which would be used to query an XML file.
CSE 233 Database Theory |UCSD|
Theory of databases. Theory of query languages, dependency theory, deductive databases, incomplete information, complex objects, object-oriented databases, and more. Connections to logic and complexity theory including finite model theory and descriptive complexity.
CSE 132C Database System Implementation |UCSD|
This course covers key systems topics in implementing an RDBMS: data storage, buffer management, indexing, sorting, relational operator implementations, query optimization, and transaction management and concurrency control. The implementation of newer Big Data systems such as Spark and MapReduce/Hadoop, as well as distributed NoSQL/key-value stores and in-memory RDBMSs.
CS 457 Database Management Systems |UNLV|
Concepts and structures necessary for design and implementation of a database management system. Survey of current database management systems and use of a DBMS. I learned database basics of relational algebra, tuple algebra, SQL, querying, and creating a database in mariaDB.

Machine Learning Course History

CSE 234 Data Systems for Machine Learning |UCSD|
This course covers key systems topics spanning the whole lifecycle of ML-based data analytics such as: data sourcing and preparation for ML, programming models and systems for scalable ML model building, and systems for faster ML deployment. Emerging topics such as governance, explanation, and ethics of ML systems. A major component of this course is reviewing cutting edge research papers from recent top conferences on these topics.

CS 489 Advanced Computer Science Topics (Machine Learning) |UNLV|
In this semester of CS 489, the advanced topic covered was machine learning. The tasks included creating Python programs and reports on running machine learning algorithms on a real dataset from UCI Machine Learning Repository’s online archive. The following programs were created: OLS, Gradient Descent, Logistic Regression, KNN, MLP Classifier, and MLP Regressor.
CS 458 Introduction to Data Mining |UNLV|
This course covered the mathematical background and application of Data Mining and Machine Learning topics. The tasks covered in this course include: computing and plotting of linear regression and gradient descent with Python programming, implementing a hidden markov model in Python, and implementing machine learning tools to find patterns such as good movie reviews and acronoyms and their definition in large datasets.
